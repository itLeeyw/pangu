<h1 align="center">盘古</h1>
<p align="center">把难看的项目梳理成关联图（文件级/函数级）。</p>


使用

```bash
$ git clone git@github.com:itLeeyw/pangu.git
$ cd ./pangu/bin
$ npm install
$ node index.js cleave -p 你的项目路径
```
